# Admin Dashboard

A professional React Admin Dashboard with Tailwind CSS and Recharts.

## 🚀 Live Demo

**URL:** https://mohamedashraf71.github.io/Admin-Dashboard/

## 📊 Features

- ✅ Charts (analytics) - Recharts
- ✅ Users Management (CRUD)
- ✅ Data Table (pagination + search + sort)
- ✅ Permissions (admin / user roles)

## 🛠️ Tech Stack

- React 18
- Tailwind CSS
- Recharts
- Vite
- Lucide React Icons

## 🏃‍♂️ Local Development

```bash
npm install
npm run dev
```
